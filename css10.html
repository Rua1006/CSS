CSS transition property

트랜지션 속성(transition property) : transition-duration(트랜지션 실행시간), transition-property(트랜지션 속성), transition-timing-function(트랜지션 시간 함수)



transition-duration : 필수 속성으로 트랜지션의 작동시간을 지정할 때 사용하는 속성으로 지정시에는 시간단위 값을 지정해야함

선택자{ transition-duration:2s; }



transition-delay : 선택 속성으로 트랜지션의 시작 전 지연시간을 의미하며, 지정한 시간 단위 값 만큼 기다린 후 작동함

선택자{ transition-duration:2s; transition-delay:0.6s; }

transition-property : 트랜지션을 적용할 CSS 속성을 지정할 수 있으며, 여러 개의 CSS 속성을 지정할 경우 transition-duration이나 transition-delay 등을 달리 적용할 수 있다.

선택자{ transition-duration:2s, 1s, 4s, 3s; transition-delay:0.6s, 0.2s, 0.1s, 0.5s; transition-property:color, background, height, border}



transition-timing-function : 트랜지션의 가속 및 감속 효과를 지정

ease | ease-in | ease-out | ease-in-out | linear | cubic-bezier()

선택자1 { transition-duration:4s; transition-timing-function:ease; }

선택자2 { transition-duration:4s; transition-timing-function:ease-in; }

선택자3 { transition-duration:4s; transition-timing-function:ease-out; }

선택자4 { transition-duration:4s; transition-timing-function:ease-in-out; }

선택자5 { transition-duration:4s; transition-timing-function:linear; }

선택자6 { transition-duration:4s; transition-timing-function:cubic-bezier(.13,1.1,1,-0.15); }
